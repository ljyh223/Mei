# Mei

一个网易云第三方音乐软件，软件仍然在开发中。但现阶段有很多bug和功能需要完善。

现在可以自己填写自己Cookie自行使用了, 只需要将Cookie中的MUSIC_U字段填到设置里面就行了

不过现在软件性能可能有点差，也可能来不及优化了。

因为上游项目除了一些问题，所以已经自写api实现了。
